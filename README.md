# Al-Mustashar
مشروع Valdis ISX: منصة ذكاء اصطناعي متطورة لتحليل سوق العراق للأوراق المالية. يعتمد النظام على محرك Kronos المطور لتقديم توصيات استثمارية مبنية على دمج التحليل الفني، الأساسي، وعوامل الاقتصاد الكلي كالنفط والدولار. يوفر المشروع مساعداً ذكياً (RAG Copilot) وواجهة عصرية لإدارة المحافظ، مع الالتزام الكامل بحقوق المصدر المفتوح للمطورين الأصليين.
